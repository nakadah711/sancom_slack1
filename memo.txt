■src
対応ソース
・CommonUtill.java
・ConvertR2K.java
　⇒「ローマ字」から「ひらがな」への変換するメイン処理
・ConvertR2KUtill.java
　⇒変換に使用するテーブル等
・SampleR2K.java
　⇒各クラスの使用例のようなモノ
・TextfileRead.java
　⇒テキストファイルからの読込処理
・TextfileWrite.java
　⇒テキストファイルの出力処理

■使用方法
・ビルド
　javac SampleR2K.java

・使用例
※ビルドしたフォルダ配下にtempフォルダを作成し、
　そこにin.txt（取り込みファイル）を配置し、out.txtとして出力する場合
java SampleR2K temp\in.txt temp\out.txt
